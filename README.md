# 📊 Stack Overflow Developer Survey 2024 - Data Analysis 🚧 [WIP]

This repository contains a statistical data analysis project conducted on the "Stack Overflow Developer Survey 2024" dataset, prepared for the Introduction to Statistics and Data Analysis course. The project is currently a Work in Progress (WIP).

## 🎯 Project Goal
To statistically analyze trends in the software world, specifically the relationships between developer types (`DevType`), programming languages used (`LanguageHaveWorkedWith`), and work models (`RemoteWork`).

## 📈 Current Status & Visualizations
Data preprocessing has been completed, and a total of **6 fundamental charts** have been created following academic standards:
* **Bar Chart & Pie Chart:** Analysis of categorical data (e.g., Remote Work distribution).
* **Histogram:** Examining the distribution of continuous variables.
* **Boxplot:** Identifying outliers and distributions based on developer types or languages.

*(Note: Initial attempts with Scatter and Heatmap charts were replaced with the current charts as they better suit the nature of the focused categorical data.)*

## 📝 Project Report (LaTeX)
The methodology and statistical findings of the project are documented in an academic, passive tone using **LaTeX**. The current version of the report in PDF format can be found in the `report/` directory.
